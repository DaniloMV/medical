/* DESCRIPCIÓN */
  
	Es un sistema con el cual es posible relizar lo siguiente:

	Registros de pacientes con fotografia. 
	Actualizacion de datos. 
	Eliminacion de datos.
	Generar archivos en formato pdf sobre la informacion de los pacientes registrados. 

	Crear y tener expedientes medicos actualizados.

	Crear historiales y registrar nuevos sintomas en cada pacientes para saber cuales enfermedades son mas comunes.
	Creacion de historial en formato pdf.

	Administracion de usuarios y medicos.

	Espero y les sea util el proyecto.

	Saludos a toda la comunidad de cesarcancino...   
  
   Version: 1.0  
   
/*   INSTALACIÓN */
   
    1.-Crear base de datos con el nombre 'paciente'
	2.-Descomprimir el archivo Paciente_database que se encuentra en mod_configuracion
	3.-Ejecutar el archivo sql en phpmyadmin
	4.-Modificar el archivo configuracion.php que se encuentra en el mod_configuracion

		$bd_host = "server";
		$bd_usuario = "usuario";
		$bd_pass = "contraseña";
		$bd_base = "base_de_datos";
		$bd_port="puerto";


	5.-Crear un usuario en http://localhost/medical/mod_admin o http://Dominio/mod_admin
	6.-Una vez creado el usuario ir a http://localhost/medical o http://Dominio/
	7.-Iniciar sesion al sistema..
	8.-Si el usuario y contraseña son correctos eliminar el directorio mod_admin...
